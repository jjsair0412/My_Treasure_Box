# overview
Shell 초기 세팅 시 사용

## 요구사항
1. OrbStack, Docker Desktop 등 설치 필요
2. 실리콘 칩(M1, M2...)맥 사용
3. iTerm2의 Font 설정 필요(Font 이름 : DejaVu Sans Mono for Powerline)

## 사용 방법
```bash
chmod +x mac_basic_setting.sh

./mac_basic_setting.sh
```
